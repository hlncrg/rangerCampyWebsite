# rangerCampyWebsite
Files for creating serving the ranger campy website

# To Run
To run the flask app, 'python application.py'

# Tweets
For tweets, place credintials in twitterCred.txt
in the form:

{"CONSUMER_KEY" : "...",
"CONSUMER_SECRET" : "...",
"TOKEN" : "...",
"TOKEN_SECRET" : "..."}
